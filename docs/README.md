## Coordination

[`AGENT_HANDOFF.md`](../AGENT_HANDOFF.md) is a temporary coordination surface
for the active implementation phase. Completed decisions belong in architecture
or roadmap documents, not in an ever-growing handoff history.

## Retired Documents

The following documents were removed because their assumptions no longer match
the project:

- root `implementation_plan.md`: described the original Single-EC2, Next.js,
  and older .NET plan;
- `technical-documentation.md`: mixed historical implementation details with
  stale framework and status information;
- `aws-terraform-infra-guide.md`: promoted the retired single-node topology;
- `mvp-deploy-gap-plan.md`: replaced by the phase-gated release roadmap.

Git history remains the source for retired documents if historical context is
needed.
